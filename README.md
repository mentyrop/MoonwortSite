# Moonwort — лендинг

Статический одностраничный лендинг проекта Moonwort. Домен: **moonwort.ru**.

## Что где

| Файл | Назначение |
|------|------------|
| `index.html` | **Продакшн-лендинг** (единственный файл, который нужен для хостинга) |
| `demo-variants.html` | Черновик: 4 варианта интерактивного блока |
| `variants.html` | Черновик: обзор ранних направлений дизайна |
| `A/B/C/D-*.html` | Ранние прототипы направлений |

Лендинг самодостаточный: один HTML-файл, шрифты подключаются с Google Fonts,
логотип и фавикон встроены как data-URI. Сборка не нужна.

## Локальный просмотр

```bash
python3 -m http.server 8742
# открыть http://localhost:8742/index.html
```
